Linux Install:

  • Download CLI driver
  • on Linux, install unixodbc and git:
apt-get install iojs npm git unixodbc unixodbc-dev -y
  • create a [BLAZING_SQUIRREL] driver in odbcinst.ini (example)
  • then:
git clone https://github.com/LeoDutra/blazing-squirrel.git
cd blazing-squirrel
npm install
npm start
  • go to http://localhost:3000

Using the FreeTDS ODBC driver

Go to node-odbc's FreeTDS driver reference
